# Jollama - Ollama Java Client

Simple Java client for the Ollama API.

## Features

- Simple API
- Chat support
- Text generation
- Minimal dependencies

## Installation

Maven

```xml
<dependency>
  <groupId>io.github.ollamaclient</groupId>
  <artifactId>ollama-java-client</artifactId>
  <version>0.0.1</version>
</dependency>
